# 📦 API RESTful de Gerenciamento de Produtos

API RESTful desenvolvida em Python utilizando Flask, SQLAlchemy e Pydantic, seguindo os padrões de arquitetura em camadas (Routes, Services, Repositories, Schemas e Models).

## 🚀 Tecnologias Utilizadas

- **Python 3.12+**
- **Flask** (Microframework web)
- **Flask-SQLAlchemy** (ORM)
- **Pydantic** (Validação de schemas)
- **Flasgger / Swagger UI** (Documentação interativa)

## 📋 Funcionalidades (CRUD Completo)

- `GET /api/products` - Listar todos os produtos
- `POST /api/products` - Criar um novo produto
- `GET /api/products/<id>` - Buscar produto por ID
- `PUT /api/products/<id>` - Atualizar um produto
- `DELETE /api/products/<id>` - Remover um produto
